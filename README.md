Search & Filter Buses - Find buses by route, date, and time
JWT Authentication - Secure login & registration with bcrypt password hashing
Interactive Seat Selection - Visual seat map with real-time availability (available, booked, counter seats)
Booking Management Dashboard - View & manage past & upcoming bookings
Email Notifications - Booking confirmations sent via Nodemailer
Payment Processing - Multi-step payment with fare, convenience fee & bank charges
Admin Fleet Management - Add/remove/manage buses with seat counts, layout, comfort levels (AC), amenities
Schedule Management - Operators manage bus routes and departure/arrival times
Staff Management - Admin controls for conductor & staff assignments
Responsive UI - Full React + Tailwind CSS frontend with Redux state management, dual-role access (user/operator)
Tech Stack: React.js, Node.js, Express, MongoDB/PostgreSQL, Redux Toolkit, JWT, bcrypt, Axios
